# Netflix (netflix)
Netflix is a Fortune 500 company that provides developer APIs and integration capabilities through its developer platform. Netflix provides partner APIs for content delivery network integration and device certification. The Open Connect program enables ISPs and device manufacturers to optimize Netflix streaming delivery.
